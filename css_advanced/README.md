# Smile School — HTML/CSS Demo

Simple static landing page built with HTML (mock content and local images). Designed as a practice project for advanced CSS layout and component structure.

## Features

- Header with navigation and logo
- Hero section with CTA
- Instructor showcase
- Testimonials
- Popular tutorials grid with star ratings
- Membership section and FAQ
- Footer with social icons (Font Awesome)

## Technologies

- HTML5
- (Optional) CSS — add your own stylesheet at `css/style.css`
- Font Awesome (CDN included in the HTML)

## Notes

- Font Awesome is loaded via CDN in the head of `index.html`. Internet connection required for icons.
- Replace placeholder images (current external fakeimg and `images/*.png`) with your own assets as needed.
- Add a stylesheet (e.g., `css/style.css`) and link it in the `<head>` to style the page.

## Suggested file structure

/

- index.html
- images/
  - Bitmap.png
  - Bitmap (1).png
  - ...
- css/
  - style.css (optional)
